Funcionamineto de la WEB

La web funciona mediante servidores que almacenan y distribuyen datos a través de direcciones IP únicas. Para facilitar la navegación, el DNS asocia nombres de dominio con direcciones IP. La comunicación entre los servidores y los navegadores se realiza mediante el protocolo HTTP.

  • Servidor: una computadora especializada que almacena y distribuye datos a través de la web.
  • Dirección IP: una serie única de números que identifica a cada dispositivo conectado a Internet.
  • DNS (Sistema de Nombres de Dominio): una base de datos gigante que asocia nombres de dominio (como "google.com" o "facebook.com") con direcciones IP.
  • HTTP (Protocolo de Transferencia de Hipertexto): el lenguaje que utilizan los servidores y los navegadores para comunicarse y transferir datos de la web. Es el protocolo principal que se utiliza para la comunicación entre clientes y servidores en la web.
